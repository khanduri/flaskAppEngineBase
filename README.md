Documentation
-------

Getting Started:
Window 1:
 1. git clone https://github.com/khanduri/flaskAppEngineBase.git
 1. initial_setup.sh
 1. dev_appserver.py src/
Window 2:
 1. cd src-fe/
 1. grunt serve

Deploy
 1. appcfg.py update src-be/

New package dependency:
 1. pip install <package>
 1. update_packages.sh
